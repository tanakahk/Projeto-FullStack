## Frontend project of a card market
Project made to practice the knowledge from the Ingate full stack WEB development course

## Build and Run
Install vue-cli:
```
npm install -g @vue/cli
```
Install all dependencies:
```
npm i
```
Run:
```
npm run serve
```
